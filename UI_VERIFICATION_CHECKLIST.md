## 기능 테스트

### Dashboard
1. [ ] 방 카드의 토글을 켜고 저장 → runtime.json 업데이트 확인
2. [ ] 최근 메시지가 1초마다 자동으로 갱신되는지 확인 (깜빡임 없어야 함)
3. [ ] Recent Activity 섹션이 실시간으로 갱신되는지 확인

### Templates
1. [ ] New Template 버튼 클릭 → 새 템플릿 생성 확인
2. [ ] Edit 버튼 클릭 → 편집 화면으로 전환
3. [ ] 카테고리 변경 후 저장 → 템플릿 파일 위치 변경 확인
4. [ ] 메시지 내용 변경 → 카카오톡 미리보기 실시간 업데이트 확인
5. [ ] Test Send 클릭 → SAFE_MODE 경고만 표시, 실제 전송 안 됨 확인
6. [ ] 삭제 버튼 클릭 → 템플릿 삭제 확인
