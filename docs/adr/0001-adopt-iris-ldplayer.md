# ADR 0001: LDPlayer + IRIS 채택

- 상태: 폐기됨 (2025-10-28, ADR 0002로 대체)
- 문맥: PC KakaoTalk UI 자동화는 스크롤/OCR 지연과 포커스 문제로 요구사항을 충족하지 못함.

## 결정
LDPlayer(안드로이드 에뮬레이터) + IRIS 이벤트 프레임워크를 기반으로 모든 카카오톡 자동화를 구현한다.

## 근거
- 메시지·입장 이벤트를 문자열로 직접 수신할 수 있어 OCR 불필요.
- 명령어 처리/방송 등 기존 커뮤니티 사례가 풍부(ponyobot, iris_bot 등).
- 다중 이벤트 스트림을 빠르게 처리할 수 있으며, 명령어 응답 지연을 수 초 이내로 유지할 수 있음.

## 결과
- PC UIA 접근 금지. 관련 스크립트/코드는 제거 대상.
- LDPlayer/IRIS 설치, 버전 관리, 인스턴스 운영 문서를 작성해야 함.
- 멀티 인스턴스 전략(필요 시)과 계정 정책 준수 계획을 별도로 수립.
