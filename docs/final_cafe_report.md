## 🔧 기술적 구현 제안

### 1. 프로젝트 구조
```
src/
├── bot/
│   ├── main.py          # 메인 실행 파일
│   ├── handlers/        # 메시지 핸들러
│   └── commands/        # 명령어 처리기
├── services/
│   ├── iris_client.py   # IRIS 클라이언트
│   ├── message_store.py # 메시지 저장소
│   └── response_engine.py # 응답 엔진
├── config/
│   ├── settings.py      # 환경 설정
│   └── templates/       # 응답 템플릿
└── tests/
    └── unit/           # 단위 테스트
```

- **Database**: SQLite (개발), PostgreSQL (운영)
- **Message Queue**: Redis
- **Monitoring**: Prometheus + Grafana
- **Logging**: Python logging + ELK Stack
